Aplikasi Monitoring Suhu dan Kelembapan Pada Ruang Server


Sistem: Aplikasi Monitoring Suhu dan Kelembapan Pada Ruang Server PT. Grahaparama Santosa Menggunakan Mikrokontroler Arduino Uno, DHT11, dan DFRduino Ethernet Shield W5100
Instansi: PT. Grahaparama Santosa
Di Buat: Tahun 2014 (Tugas Akhir)
Peran: Developer

Aplikasi ini saya buat pada tahun 2014, dengan bahasa Pemrograman JAVA dan dengan modul atau alat pembantu menggunakan mikrokontroler Arduino Uno, DHT11, dan DFRduino Ethernet Shield W5100.

Aplikasi ini di buat dengan tujuan dapat memantau suhu dan kelembapan pada ruang server dengan memberikan informasi secara realtime pada layar computer serta sebagai bahan penilaian Tugas Akhir saya. Aplikasi atau sistem ini berfungsi memberikan peringatan berupa alarm dan norifikasi berupa email kepada Administrator server apabila suhu atau kelembapan berada di luar batas serta mengaktifkan AC cadangan secara otomatis untuk menjaga kestabilan ruang server.
